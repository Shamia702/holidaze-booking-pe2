# Holidaze — Accommodation Booking App

A dual facing accommodation booking application built for the Noroff PE2 exam. Customers can browse and book venues while venue managers can create and manage their listings.

## Built With

- React + Vite
- Tailwind CSS
- React Router v6
- Noroff Holidaze API v2

## Features

### All Users

- Browse all venues with search and pagination
- View single venue pages with calendar
- View available and booked dates on calendar
- Register as a Customer or Venue Manager

### Customers

- Log in and Log out
- Book a venue
- View upcoming and past bookings
- Cancel a booking
- Update profile avatar

### Venue Managers

- Log in and out
- Create, edit and delete venues
- View all bookings for their venues
- Update profile avatar

## Getting Started

### Installation

```bash
git clone https://github.com/Shamia702/holidaze-booking-pe2
cd holidaze-booking-pe2
npm install
```

### Environment Variables

Create a `.env` file in the root:

VITE_API_KEY=your_noroff_api_key

### Run locally

```bash
npm run dev
```

### Build

```bash
npm run build
```
